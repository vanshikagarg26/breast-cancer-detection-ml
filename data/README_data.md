# Dataset

The histopathological image dataset used in this project is the 
Invasive Ductal Carcinoma (IDC) dataset.

Images are organised into two classes:
- `yes/` — Malignant (cancerous)
- `no/` — Benign (non-cancerous)

Dataset not uploaded due to size. Available on Kaggle:
https://www.kaggle.com/datasets/paultimothymooney/breast-histopathology-images
